# clps-jnr-software-engr-assessment
Answers for the CLPS Junior Software Engineering role - 19 Jan 2026

## Files
- `assessment_1.py`: Assessment 1
- `assessment_2.py`: Assessment 2

## Running the tests for the assessments
To run the tests, pytest is used in this project.

To install pytest:
```
pip install -U pytest
```
To run the tests:

```
pytest test_a1.py
```

where `a1` refers to Assessment 1, and `a2` refers to Assessment 2.
